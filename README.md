**Yadro Telecom test task for C++ intership**

1) Go to */build* directory in terminal
2) Run ```cmake -G "MinGW Makefiles" ../CmakeLists.txt``` command
3) Run ```make``` command
4) Executable (.exe) file will appear in */build* directory
